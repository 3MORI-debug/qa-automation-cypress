# QA Automation - Cypress

## 📌 Sobre o projeto
Este projeto contém testes automatizados utilizando Cypress para validar a funcionalidade de login do site SauceDemo.

## 🧪 Cenários testados
- Login com sucesso
- Login com senha inválida
- Login com campos vazios

## 🛠️ Tecnologias utilizadas
- Cypress
- JavaScript
- Node.js

## ▶️ Como executar
```bash
npm install
npx cypress open
