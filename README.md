# 🧪 QA Automation - Cypress

## 📌 Sobre o projeto
Projeto de automação de testes utilizando Cypress para validar o fluxo de login do site SauceDemo.

---

## 🚀 Tecnologias utilizadas
- Cypress
- JavaScript

---

## 🧠 Estratégia de Testes
Os testes cobrem:

- Cenários positivos (login válido)
- Cenários negativos (credenciais inválidas)
- Validação de mensagens de erro
- Testes independentes

---

## 🏗 Estrutura do projeto

cypress/
 ├── e2e/
 ├── support/
 ├── fixtures/

---

## 🔁 Boas práticas aplicadas

- Uso de beforeEach
- Comandos customizados
- Separação de cenários
- Testes isolados
